11. Summary

Concept Purpose
CSS Styles HTML elements
Selectors Select elements to style
Specificity Determines which style wins
Colors Change text/background colors
Backgrounds Add colors/images behind content
Borders Create element outlines
Margins Space outside elements
Padding Space inside elements
