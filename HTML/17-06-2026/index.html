Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. HTML provides the basic structure of a webpage using elements and tags.

A web page can contain headings, paragraphs, images, links, tables, forms, and many other elements.

Summary

  • HTML is used to create web pages.
  • Block elements start on a new line.
  • Inline elements stay within the same line.
  • Attributes provide extra information to elements.
  • An element consists of tags and content.
  • HTML provides six levels of headings.
